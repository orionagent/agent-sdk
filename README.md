# Agent SDK v0.2.2 AAR包

# 概述

> Agent SDK 是一个用于机器人交互的Android开发套件，提供了应用与机器人Agent服务进行通信的能力。SDK支持应用级和页面级的Agent开发，可以实现自然语言交互、动作规划和执行等功能。

* JDK版本: Java 11
* Kotlin、Java语言开发
